Jotform API - Issue Demo

Summary

Using the Jotform JavaScript SDK against Jotform.com works fine.

However, using it against forms.bgsu.edu fails with CORS errors.

Called outside a browser context (no CORS), the API works fine for forms.bgsu.edu.

Examples

We have two nearly identical pages designed to test the Jotform JavaScript SDK.

  • JotForms.com Example
  • Forms.bgsu.edu Example

Both examples prompt the user to authenticate and authorize the app. This appears to work successfully.

Next, they try to fetch user information over the API. While the jotform.com variant works, the forms.bgsu.edu variant fails with CORs errors. This indicates that even though the application has been approved, the origin is not being allowed.

What is CORS

Cross-Origin Resource Sharing (CORS) is a security mechanism that allows web browsers to access resources from other domains. CORS is used to prevent malicious websites from accessing data on other sites without permission. Read more on CORS
